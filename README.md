# HomingMissilesPrototype

A mobile survival prototype built in Unity where the player controls a rocket, avoids enemies, and uses enemy behavior against them.

## Gameplay Overview

- Control a rocket using a mobile joystick
- Collect coins placed in world space
- Avoid homing missiles that track the player
- Survive against spawning mothership enemies
- Use smart movement to redirect homing missiles into motherships

## Core Mechanic

The unique twist is indirect combat:
Instead of directly attacking enemies, the player manipulates movement so that homing missiles collide with motherships and destroy them.

## Built With

- Unity (C#)
- Mobile Joystick Input System
- Spawning System
- Physics-based collision interactions

## Platform

- Android (Prototype Build)

## Features

- Smooth mobile movement controls
- Enemy homing missiles
- Dynamic enemy spawning
- Risk-reward coin collection system
- Emergent gameplay interactions

## Goal of Prototype

This project was built to explore:
- Mobile control feel
- Homing missiles player follow system
- Emergent gameplay design
- Player-driven enemy interaction mechanics
